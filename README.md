React Boilerplate
=====================

A minimal and light dev environment for ReactJS.


Install the dependencies and start the server.

```
npm install
npm start
open http://localhost:3000
```

### Linting

This boilerplate project includes React ESLint configuration.

```
npm run lint
```

### Dependencies

* babel-core
* babel-loader
* babel-preset-es2015
* babel-preset-react

* node-sass
* sass-loader
* sockjs-client
* style-loader
* webpack
* webpack-dev-server
